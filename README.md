# Computer-Vision-Project-NFL-Semerano-Alexa
Computer Vision Project based on NFL 1st and Future challenge about helmet impact detection: 
https://www.kaggle.com/competitions/nfl-impact-detection/overview

The player_tracking file contains the testing procedure of our tracking method applied on the helmets data from the dataset csv

The Training_R(2+1)D file contains the training procedure of our R(2+1)D model

The Test_R(2+1)D file contains the testing procedure on 6 videos taken from the dataset of our R(2+1)D model

The tracking_results folder contains three different tracking methods applied on the same video, we reached a better performance with a custom method

The result_videos folder contains the 6 resulting videos provided by the execution of yhr Test_R(2+1)D file
